Repository for scripts used in:

Integrated in-silico and experimental analysis identifies bromocriptine and lifitegrast as OTUB1-targeting compounds in B-cell acute lymphoblastic leukemia 

Contents:

OTUB1 expression/Biomarker analysis
- TCGAbiolinks
- DESeq2
- Dispersion, MA and PCA plot
- Volcano plot
- Heatmap
- clusterProfiler
- svaseq correction
- FDR correction of survival data
- Univariate Cox regression
- Multivariate Cox regression

Molecular_Dynamics/
- GROMACS molecular dynamics simulations
- MM/PBSA calculations
